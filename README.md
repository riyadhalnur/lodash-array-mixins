# lodash-array-mixins
A collection of lodash mixins to do fancy stuff with Arrays  

## Methods  
`_.randElem([])`  
Returns a random element in an array  

`_.largestElem([])`  
Returns the size of the largest element in an array (most suitable for an array of strings)
